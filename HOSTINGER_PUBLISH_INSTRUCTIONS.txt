# 🚀 Hostinger Publishing Guide - I AM CYBER SOLDIER

Since your website is built with Next.js (a modern React framework), there are two ways to publish it on Hostinger. 

---

### 🟢 RECOMMENDED: Option 1 - The "Professional" Way (Vercel + Hostinger Domain)
Next.js works best on Vercel (the creators of Next.js). This setup is FREE, much faster, and supports your AI Voice Assistant perfectly.

1. **Create a GitHub Account** (if you don't have one).
2. **Push your code to GitHub:**
   - Open your terminal and run:
     ```bash
     git init
     git add .
     git commit -m "Deployment ready"
     # Create a new repository on GitHub and run the 'git remote add' and 'git push' commands they provide.
     ```
3. **Connect to Vercel:**
   - Go to [vercel.com](https://vercel.com) and log in with GitHub.
   - Click "Add New" -> "Project" and import your repository.
4. **Add your AI Key:**
   - In Vercel, under "Environment Variables", add:
     - Name: `GROQ_API_KEY`
     - Value: `YOUR_GROQ_API_KEY_HERE`
   - Click "Deploy".
5. **Connect your Hostinger Domain:**
   - In Vercel Project Settings -> Domains, type `iamcybersoldier.com`.
   - Vercel will give you two "DNS Records" (A record and CNAME).
   - Log in to **Hostinger hPanel** -> Select Domain `iamcybersoldier.com` -> DNS / Nameservers.
   - Replace the existing A record and CNAME record with the values provided by Vercel.

---

### 🟡 Option 2 - The "Hostinger Shared Hosting" Way (Static Version)
Use this if you *must* keep everything inside the Hostinger File Manager. 
**Note:** Your AI Voice Assistant might not work in this mode unless we make technical changes, as standard Hostinger "Single" plans don't run Next.js "servers".

1. **Prepare the code for Static Export:**
   - Open `next.config.mjs` and add `output: 'export',` after `typescript: { ignoreBuildErrors: true },`.
2. **Build the website:**
   - Run this command in your terminal:
     ```bash
     npm run build
     ```
   - This will create a folder named `out` in your project directory.
3. **Upload to Hostinger:**
   - Log into **Hostinger hPanel** -> Hosting -> Manage.
   - Go to **File Manager** -> `public_html`.
   - **IMPORTANT:** Delete the existing WordPress files (wp-admin, wp-content, etc.) or move them to a backup folder.
   - Upload everything INSIDE the `out` folder directly into `public_html`.
4. **Verify:**
   - Visit `www.iamcybersoldier.com` to see your new site live!

---

### ⚠️ IMPORTANT: WordPress Migration
Since you currently have WordPress installed, uploading the new site will REPLACE it. 
If you have blog posts or pages on WordPress that you want to keep, you should copy that text now before deleting the WordPress files on Hostinger.
